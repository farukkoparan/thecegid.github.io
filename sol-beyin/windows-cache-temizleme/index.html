Windows Cache Temizleme

Bilgisayarlar kurulum ardından zaman içerisinde önbellek verileri, oturum bilgileri ve çerez gibi sisteminizi yavaşlatan dosyalarla dolup taşar. Bilgisayarınızı yavaşlatan tek şey bu da değildir, eski sürücüler de sisteminizin geride kalmasını sağlar. Bu gibi durumlarda kullanıcıya yardım eden birkaç program vardır. Kurulum sonrasında bilgisayarınıza kurmanız gereken temel programlar aşağıdaki gibidir:

Performans için gerekli olanlar;

https://www.iobit.com/en/driver-booster - sürücü güncelleme

https://www.ccleaner.com/ccleaner - önbellek temizleme

https://www.malwarebytes.com/adwcleaner - performans iyileştirme

https://get.adobe.com/shockwave - web oynatıcısı (.dcr)

https://get.adobe.com/tr/flashplayer - web oynatıcısı (.swf)

Bu programlara ek olarak windows 10 update önbelleği temizleme yöntemi kullanılabilir. Win + R 'ye basarak çalıştır menüsünü açarak arama kutusuna cmd yazıp Tamama tıklayın. Karşınıza çıkan Komut İstemine;

net stop wuauserv
komutunu girin.

C:\Windows\SoftwareDistribution\Download dizinine gidip tüm dosyaları silin. (bilgisayara bir zararı yoktur)

net start wuauserv
ardından yine Komut İstemi kısmına bunu yazarak işlemi tamamlayabilirsiniz.


Bu programlar yanında alternatifleri olan birkaç uygulama daha sayabilirim.
